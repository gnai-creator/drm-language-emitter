# Third-Party Licenses And Data Provenance

This file tracks bundled data and third-party artifacts that affect reuse of the repository.

## Source Code Dependencies

Declared runtime dependency:

- PyTorch, via `torch>=2.0` in `pyproject.toml`. Check the installed PyTorch distribution for its license notices.

Declared development dependencies:

- `pytest>=7.0`
- `pyyaml>=6.0`

## Bundled Data

### `data/wikipedia_en_20231101_sample.txt`

Purpose: optional local training/evaluation text for the 125M/350M scale comparison.

Provenance: generated by `scripts/prepare_wikipedia_en.py` from Hugging Face `wikimedia/wikipedia`, default config `20231101.en`, split `train`.

License: Wikipedia text is generally distributed under Creative Commons Attribution-ShareAlike terms, with historical GNU Free Documentation License considerations. Verify the current upstream dataset card and Wikimedia licensing terms before redistributing generated samples.

Notes: this file is generated locally and should normally remain outside git unless a small, reviewed sample is intentionally versioned.

### `data/tiny.txt`

Purpose: tiny local smoke-test corpus for training and generation workflows.

Provenance: project-authored text included in this repository.

License: covered by the repository license unless superseded by a more specific notice.

Notes: not a benchmark dataset and not suitable for model-quality claims.

## Generated Benchmark Artifacts

Files under `docs/benchmarks/` are generated experiment summaries, charts, dashboards, and tables from local project runs.

Provenance: generated by scripts in this repository.

License: covered by the repository license unless an individual artifact states otherwise.

## External Documents

The repository contains `DRM__Directional_Relational_Manifolds_V1_1 (1).pdf`. Keep its provenance and reuse rights clear before redistributing derived materials.

## Maintenance Rule

When adding a new dataset, model checkpoint, pretrained artifact, image, or external document, add:

- path;
- source URL or author;
- license;
- whether redistribution is allowed;
- whether it may be used for training, evaluation, or documentation only.
